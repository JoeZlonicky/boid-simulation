# IK Demo
My progress making an inverse kinematics demo with OpenGL.

## Running from CLI
1. `cmake -B build`
2. `cmake --build build --target ik-demo`
3. `cd build`
4. `./ik-demo`

## Testing from CLI
1. `cmake -B build`
2. `cmake --build build --target ik-demo-tests`
3. `cd build`
4. `./ik-demo-tests`
